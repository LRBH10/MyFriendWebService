--- --- --- --- --- --- --- --- --- --- --- ---  Action : login  --- --- --- --- --- --- --- --- --- --- --- --- 
  
* action=login *
    Create user on the server and returned the token if success

This module requires read rights
Parameters:
  pseudo          - (requiered) the pseudo name associated to the user
  password        - (requiered) the password associated to the user
  
Example :
  api.php?action=login&pseudo=bibouh13&password=rabah123
  api.php?action=login&pseudo=bibouh13@live.fr&password=rabah123

--- --- --- --- --- --- --- --- --- --- --- ---  Action : createuser  --- --- --- --- --- --- --- --- --- --- --- --- 
  
* action=createuser *
    Create user on the server and returned the token if success

This module requires read rights
Parameters:
  pseudo          - (requiered) the pseudo name associated to the user
  password        - (requiered) the password associated to the user
  firstname       - (optional) the first name of the user
  lastname        - (optional) the last name of the user

Example :
        To create a user with pseudo 'bibouh123' and password 'rabah123' : 
    api.php?action=createuser&pseudo=bibouh123&password=rabah123
        To create a user with pseudo 'bibouh123' and password 'rabah123' and firstname  'rabah' and last name 'LAOUADI' : 
    api.php?action=createuser&pseudo=blazo123&password=nastov123&firstname=blazo&lastname=NASTOV


       To create count with mail
    api.php?action=createuser&pseudo=bibouh123@live.fr&password=rabah123
